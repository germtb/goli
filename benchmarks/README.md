# Benchmarks

This directory contains performance benchmarks for goli and comparisons against other terminal UI frameworks.

## Running Benchmarks

### goli CPU Usage

```bash
go run ./cpu/main.go
```

### Comparison with Ink (React for terminals)

```bash
cd comparison-ink
./run.sh
```
